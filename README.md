# Elegance - Fine Dining Restaurant Website

A modern, elegant restaurant website built with HTML, CSS, and JavaScript.

## 📁 Files Included

- **index.html** - Main home page with hero section, about us, featured dishes, and contact form
- **menu.html** - Complete restaurant menu with appetizers, mains, desserts, and beverages
- **gallery.html** - Photo gallery with modal lightbox viewer
- **styles.css** - Professional styling with responsive design
- **script.js** - Interactive features (navigation, smooth scrolling, animations)
- **README.md** - This file

## ✨ Features

### Home Page
- Elegant hero section with call-to-action button
- About the restaurant with statistics
- Featured dishes showcase
- Contact information and form
- Smooth navigation with active link indicators

### Menu Page
- Organized menu sections (Appetizers, Mains, Desserts, Beverages)
- Detailed descriptions and pricing for each dish
- Professional layout with elegant styling

### Gallery
- Image gallery grid
- Hover effects with captions
- Modal lightbox viewer
- Keyboard navigation (arrow keys to navigate, ESC to close)
- Fully responsive

### Interactive Features
- Smooth scrolling navigation
- Mobile-responsive hamburger menu
- Form validation
- Scroll animations
- Active navigation indicators
- Keyboard shortcuts in gallery

## 🚀 How to Use

1. Open `index.html` in your web browser
2. Navigate through different sections using the navigation menu
3. Visit the Menu page to see the full dining options
4. Browse the Gallery with interactive image viewer
5. Fill out the contact form to send messages

## 📱 Responsive Design

The website is fully responsive and works on:
- Desktop browsers
- Tablets
- Mobile phones

Mobile menu automatically converts to a hamburger menu on smaller screens.

## 🎨 Customization

### Update Restaurant Information
- Edit the restaurant name, address, phone, and hours in the relevant HTML files
- Update social media links in the footer

### Change Images
- Replace image URLs in the HTML files with your own images
- Images are currently sourced from Unsplash

### Modify Menu
- Edit menu items, descriptions, and prices in `menu.html`
- Add or remove menu sections as needed

### Adjust Colors
- Edit CSS variables in `styles.css`:
  - `--primary-color`: Main dark color (#1a1a1a)
  - `--secondary-color`: Gold accent color (#d4af37)
  - `--accent-color`: Light color (#f5f5f5)

## 🔧 Technical Details

- Built with semantic HTML5
- CSS Grid and Flexbox layouts
- Vanilla JavaScript (no frameworks)
- Intersection Observer API for animations
- Mobile-first responsive design

## 📝 Notes

- This is a static website - contact forms won't send emails without backend setup
- Images are loaded from external URLs (Unsplash)
- For production, replace placeholder images with your own photos
- Consider adding a backend service for the contact form functionality
